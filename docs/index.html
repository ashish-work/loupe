Claude Code plugin

loupe

Interactive, human-in-the-loop PR review. Claude finds the issues; you curate them in a browser — then the curated review goes back to post.

View on GitHub Release v1.0.0

Install

# in Claude Code
/plugin marketplace add ashish-work/loupe
/plugin install loupe@loupe

Then just ask Claude to review a PR, or run /loupe. Requires Python 3.8+ (stdlib only); gh CLI optional for GitHub PRs.

What you get

Curated findings

Claude proposes severity-tagged, line-anchored comments; you accept, edit, or dismiss each — and add your own. A few strong findings beat a wall of nitpicks.

Overview diagrams

Mermaid architecture overlay, capability map, or user-journey flow rendered above the diff, so you grasp the idea of the change before the line diff. Claude asks which to include.

UML structure map

For type-shape changes — added / modified / removed classes, interfaces, relationships — color-coded and clickable straight into the diff.

One clean loop

Submit, and the curated review (kept comments + your own + verdict) returns to Claude as JSON to post a GitHub review or summarize.

Examples

“review PR #482”

Claude fetches the diff, writes findings, opens the loupe UI. You accept/edit/dismiss, pick Approve / Comment / Request changes, and it posts a GitHub review.

“review this branch before I merge”

No PR yet — loupe builds the diff from git diff main...HEAD and runs the same review loop locally.

“/loupe” on a big feature PR

Claude offers overview diagrams (architecture / capability / journey), renders them as Mermaid above the diff, and you review with the meta-picture in hand.

“look at this diff and tell me what's risky”

loupe surfaces only what you keep — curate in the browser instead of scrolling a chat dump.

How it works

  1. Claude reads the PR/diff and writes findings (+ optional UML structure and Mermaid diagrams).
  2. loupe — a zero-dependency Python server — renders a self-contained UI and opens your browser.
  3. You curate: accept / edit / dismiss, add line comments, pick a verdict, hit Send review.
  4. The curated review returns to Claude as JSON to post to GitHub or summarize.
